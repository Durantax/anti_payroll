# 📋 근로소득 간이세액표 계산 가이드

## 📌 간이세액표란?

근로소득 간이세액표는 **월 급여 지급 시 원천징수할 소득세**를 간편하게 조회할 수 있도록 국세청에서 제공하는 표입니다.

**중요**: 이 표의 세액은 **이미 여러 공제를 반영한 최종 납부세액**입니다!

---

## 🔍 간이세액표 산출 원리 (2024.2.29 개정 기준)

간이세액표는 다음 **6가지 공제**를 모두 계산하여 반영한 값입니다:

### 1️⃣ 근로소득공제 (소득세법 제47조)
총급여액에 따라 일정 비율을 공제합니다.

| 총급여액 | 공제액 |
|---------|--------|
| 500만원 이하 | 총급여액의 70% |
| 500만원 초과 ~ 1,500만원 이하 | 350만원 + 500만원 초과액의 40% |
| 1,500만원 초과 ~ 4,500만원 이하 | 750만원 + 1,500만원 초과액의 15% |
| 4,500만원 초과 ~ 1억원 이하 | 1,200만원 + 4,500만원 초과액의 5% |
| 1억원 초과 | 1,475만원 + 1억원 초과액의 2% |

### 2️⃣ 기본공제 (소득세법 제50조)
- **본인 및 부양가족 1인당 연 150만원** 공제
- 간이세액표의 **"공제대상 가족수"**가 이에 해당

### 3️⃣ 특별소득공제 및 특별세액공제 중 일부 (1페이지 표)

연간 총급여액과 가족 수에 따라 다음과 같이 공제합니다:

#### 📊 3,000만원 이하
- **가족 1명**: 310만원 + 총급여액의 4%
- **가족 2명**: 360만원 + 총급여액의 4%
- **가족 3명 이상**: 500만원 + 총급여액의 7%

#### 📊 3,000만원 초과 ~ 4,500만원 이하
- **가족 1명**: 310만원 + 총급여액의 4% - (3,000만원 초과액의 5%)
- **가족 2명**: 360만원 + 총급여액의 4% - (3,000만원 초과액의 5%)
- **가족 3명 이상**: 500만원 + 총급여액의 7% - (3,000만원 초과액의 5%)

#### 📊 4,500만원 초과 ~ 7,000만원 이하
- **가족 1명**: 310만원 + 총급여액의 1.5%
- **가족 2명**: 360만원 + 총급여액의 2%
- **가족 3명 이상**: 500만원 + 총급여액의 5% + (4,000만원 초과액의 4%)

#### 📊 7,000만원 초과 ~ 1.2억원 이하
- **가족 1명**: 310만원 + 총급여액의 0.5%
- **가족 2명**: 360만원 + 총급여액의 1%
- **가족 3명 이상**: 500만원 + 총급여액의 3%

### 4️⃣ 연금보험료공제 (소득세법 제51조의3)
- **국민연금** 등 연금보험료 **전액 공제**

### 5️⃣ 근로소득세액공제 (소득세법 제59조) ⭐

**💡 "근로소득자는 공제에 소득세가 없다"는 의미입니다!**

#### 📌 산출세액이 130만원 이하인 경우:
```
공제액 = 산출세액 × 55%
(단, 최저한 74만원 또는 산출세액 중 적은 금액)
```

#### 📌 산출세액이 130만원 초과인 경우:
```
공제액 = 71.5만원 + (130만원 초과분 × 30%)
```

#### 💼 예시
- **산출세액 100만원**
  - 공제: 100만원 × 55% = **55만원**
  - 실제 납부: **45만원**

- **산출세액 200만원**
  - 공제: 71.5만원 + (70만원 × 30%) = **92.5만원**
  - 실제 납부: **107.5만원**

**📌 간이세액표는 이 공제를 이미 반영한 "최종 납부세액"입니다!**

### 6️⃣ 해당 세율 적용
과세표준에 따른 누진세율 (6% ~ 45%)을 적용합니다.

## 📊 간이세액표 사용 방법

### A. 월급여액 1,000만원 이하
- **조견표(2~32페이지)** 에서 월급여액 및 가족 수에 맞는 세액 직접 조회

#### 예시: 월급 300만원, 가족 3인
```
조견표에서 300만원 구간, 가족 3인 → 31,940원
```

### B. 월급여액 1,000만원 초과 (33페이지 계산식)

**A = 1,000만원일 때의 해당 가족 수별 세액**

| 월급여액 구간 | 계산식 |
|-------------|--------|
| **1,000만원 ~ 1,400만원** | A + (초과액 × 98% × 35%) + 25,000원 |
| **1,400만원 ~ 2,800만원** | A + 1,397,000원 + (초과액 × 98% × 38%) |
| **2,800만원 ~ 3,000만원** | A + 6,610,600원 + (초과액 × 98% × 40%) |
| **3,000만원 ~ 4,500만원** | A + 7,394,600원 + (초과액 × 40%) |
| **4,500만원 ~ 8,700만원** | A + 13,394,600원 + (초과액 × 42%) |
| **8,700만원 초과** | A + 31,034,600원 + (초과액 × 45%) |

#### 예시: 월급 1,200만원, 가족 3인
```
1. A = 1,000만원, 가족 3인 → 600,000원 (조견표)
2. 초과액 = 1,200만원 - 1,000만원 = 200만원
3. 소득세 = 600,000 + (2,000,000 × 98% × 35%) + 25,000
         = 600,000 + 686,000 + 25,000
         = 1,311,000원
```

## 🧒 자녀 세액공제 (2페이지)

**8세 이상 20세 이하 자녀**가 있는 경우 추가 공제:

| 자녀 수 | 공제액 |
|--------|--------|
| **1명** | 12,500원 |
| **2명** | 29,160원 |
| **3명 이상** | 29,160원 + (2명 초과 1명당 25,000원) |

#### 예시: 자녀 4명
```
공제액 = 29,160 + (25,000 × 2) = 79,160원
```

## 👨‍👩‍👧‍👦 가족 11명 초과 시 (4페이지)

```
세액 = (11명 세액) - [(10명 세액 - 11명 세액) × 11명 초과 가족 수]
```

#### 예시: 월급 300만원, 가족 13인
```
1. 11명 세액 = 0원 (조견표)
2. 10명 세액 = 3,600원 (조견표)
3. 추가 공제 = (3,600 - 0) × 2 = 7,200원
4. 최종 세액 = 0 - 7,200 = 0원 (음수는 0원 처리)
```

## 🧮 계산 예시 (종합)

### 예시 1: 월급 300만원, 가족 3인, 자녀 1명 (8~20세)

1. **간이세액표 조회**: 300만원, 가족 3인 → **31,940원**
2. **자녀 세액공제**: 1명 → **-12,500원**
3. **소득세**: 31,940 - 12,500 = **19,440원** (1의 자리 절사)
4. **지방소득세**: 19,440 × 10% = **1,940원** (1의 자리 절사)

**최종 납부**: 소득세 19,440원 + 지방소득세 1,940원 = **21,380원**

---

### 예시 2: 월급 1,200만원, 가족 2인, 자녀 2명 (8~20세)

1. **A (1,000만원, 가족 2인)**: 770,000원
2. **초과액 계산**: 1,200만원 - 1,000만원 = 200만원
3. **소득세 (계산식)**: 770,000 + (2,000,000 × 98% × 35%) + 25,000
   = 770,000 + 686,000 + 25,000 = **1,481,000원**
4. **자녀 세액공제**: 2명 → **-29,160원**
5. **소득세**: 1,481,000 - 29,160 = **1,451,840원** (1의 자리 절사)
6. **지방소득세**: 1,451,840 × 10% = **145,180원** (1의 자리 절사)

**최종 납부**: 소득세 1,451,840원 + 지방소득세 145,180원 = **1,597,020원**

## 📌 핵심 포인트

### ✅ 간이세액표는 "최종 납부세액"
- 근로소득공제, 기본공제, 특별소득공제, 연금보험료공제, **근로소득세액공제**가 모두 반영됨
- 표에서 조회한 값 그대로 원천징수하면 됩니다

### ✅ 근로소득세액공제 덕분에 실제 세금이 줄어듭니다
- 산출세액의 최대 55%까지 공제 가능
- "근로소득자는 공제에 소득세가 없다"는 의미!

### ✅ 자녀가 있으면 추가 공제
- 8~20세 자녀 1명당 최소 12,500원 ~ 최대 25,000원 공제

### ✅ 가족이 많을수록 세금이 줄어듭니다
- 가족 수가 많을수록 간이세액표의 세액이 낮아짐
- 11명 초과 시에도 추가 공제 가능
