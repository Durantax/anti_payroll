# 🔴 월급제 직원 급여 0원 문제 해결

## 문제 증상
- 주소정근로시간: 14시간
- 정상근로시간: 50시간 입력
- 결과: 0원

## 🔍 원인 분석

### 월급제 급여 계산 로직
```dart
if (worker.salaryType == 'MONTHLY' && worker.monthlySalary > 0) {
    baseSalary = worker.monthlySalary;  // ✅ 월급을 그대로 사용
    // normalHours는 사용하지 않음!
} else {
    baseSalary = (hourlyRate * normalHours).round();  // ❌ 시급제만 사용
}
```

## 💡 해결 방법

### 확인 사항 1: 월급여 (monthlySalary) 입력 여부
월급제 직원은 **"월급여"** 필드에 금액이 입력되어 있어야 합니다.

```
✅ 올바른 입력:
- 급여 유형: 월급제 (MONTHLY)
- 월급여: 3,000,000원 ← 반드시 입력!
- 정상근로시간: (월급제는 사용 안 함)
- 주소정근로시간: 14시간 (통상시급 계산용)

❌ 잘못된 입력:
- 급여 유형: 월급제 (MONTHLY)
- 월급여: 0원 또는 비어있음 ← 문제!
- 정상근로시간: 50시간 ← 월급제는 이 값을 안 씀
```

### 확인 사항 2: 월급제 vs 시급제 차이

| 항목 | 월급제 (MONTHLY) | 시급제 (HOURLY) |
|------|-----------------|----------------|
| 기본급 계산 | **월급여 그대로** | 시급 × 정상근로시간 |
| 월급여 필드 | ✅ **필수 입력** | 사용 안 함 (0원) |
| 시급 필드 | 통상시급 자동계산 | ✅ **필수 입력** |
| 정상근로시간 | 사용 안 함 | ✅ **필수 입력** |
| 주소정근로시간 | 통상시급 계산용 | 주휴수당 판정용 |

### 월급제 기본급 계산식
```
기본급 = 월급여 (그대로)
통상시급 = 월급여 ÷ (주소정근로시간 × 4.345)

예시:
- 월급여: 3,000,000원
- 주소정근로시간: 14시간
- 통상시급 = 3,000,000 ÷ (14 × 4.345) = 49,343원

연장수당 등은 이 통상시급(49,343원)을 기준으로 계산
```

## ✅ 해결 절차

1. **직원 정보 확인**
   - 급여 유형: "월급제" 선택 확인
   - **월급여: 반드시 금액 입력!** (예: 3,000,000)
   - 주소정근로시간: 14시간 (통상시급 계산용)

2. **월별 근무 데이터 입력**
   - 정상근로시간: 입력해도 되지만 사용하지 않음
   - 연장시간, 야간시간, 휴일시간: 필요시 입력

3. **결과 확인**
   - 기본급 = 월급여 (예: 3,000,000원)
   - 연장수당 = 통상시급 × 연장시간 × 1.5
   - 주휴수당 = "월급에 포함" (14시간 < 15시간이어도 월급제는 포함)

## 🎯 핵심 정리

**월급제는 "월급여" 필드가 필수!**
- 월급여가 0원이면 → 기본급 0원
- 정상근로시간은 월급제에서 사용하지 않음
- 주소정근로시간은 통상시급 계산에만 사용

**시급제는 "시급" + "정상근로시간" 필드가 필수!**
- 기본급 = 시급 × 정상근로시간
- 주소정근로시간 < 15시간이면 주휴수당 없음
